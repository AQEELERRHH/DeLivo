// Process

Five steps from pickup to paid.

Every stage is logged on-chain. AI validates before a single token moves.

📦1

Pickup

Driver logs GPS coordinates and timestamp. Immutably recorded on GenLayer.

🗺️2

In Transit

Optional waypoints build an unforgeable audit trail of the route.

📸3

Delivery

Driver submits GPS + photo proof (IPFS/Arweave) to the contract.

🤖4

DeLivo Shield

GenLayer validators run LLM consensus: route check, anomaly scan, photo verify.

✅5

Pay / Resolve

Clean delivery → instant payment. Dispute → validator court arbitrates.

// DeLivo Shield

Every delivery, triple-verified.

Our AI verification stack runs on GenLayer's decentralised validator network — no single point of failure, no bribable arbitrator.

📍

GPS Route Analysis

LLM validators cross-examine pickup and delivery coordinates against transit time. Teleporting, impossible speeds, and reroutes are flagged instantly.

🕵️

Anomaly Detection

Unexplained long stops, destination avoidance, and GPS spoofing patterns are detected by consensus across multiple independent AI validators.

📸

Photo Proof AI

Delivery photo hosted on IPFS is retrieved and analysed. AI checks for a visible package, plausible delivery scene, and signs of image manipulation.

⚖️

Optimistic Democracy

Disputes escalate to GenLayer's validator court. Multiple LLMs deliberate and vote. The majority verdict is enforced on-chain — trustless and final.

🔐

Escrow & Auto-Pay

Payment is locked in the contract at creation. Releases automatically when AI + recipient confirm delivery. Never held by a middleman.

🗂️

Immutable Audit Trail

Every GPS log, waypoint, photo URL, and AI verdict is stored permanently on-chain. Dispute evidence is cryptographically incontestable.

🧠

LLM-Powered Validation

Intelligent Contracts call real LLMs during execution. Validators independently assess route data and photo evidence — no oracle needed.

🌐

Native Web Data Access

GenLayer contracts fetch live data from URLs directly on-chain via gl.get_webpage(). DeLivo reads photo proofs from IPFS natively.

⚖️

Optimistic Democracy

Non-deterministic outputs are resolved by multi-validator consensus. GenLayer's equivalence principle ensures agreement even when LLMs word things differently.

🐍

Python-Native Development

DeLivo contracts are written in pure Python using the GenVM SDK. No Solidity, no DSLs — full Python ecosystem for logistics logic.
